Apa itu HTML ?

Hypertext Markup Language atau HTML bahasa markup yang digunakan untuk membuat struktur halaman website.

HTML terdiri dari kombinasi teks dan simbol yang disimpan dalam sebuah file. Dalam membuat file HTML, terdapat standar atau format khusus yang harus diikuti. Format tersebut telah tertuang dalam standar kode internasional atau ASCII ASCII (American Standard Code for Information Interchange).
